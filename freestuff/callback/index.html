Quick Setup

Step one:

Tell the bot in which channel it should send the annoucements using: @FreeStuff set channel #channel with #channel being a channel of your choice, of course.

Step two:

Try it out! You can at any time try if everything works correctly by using this command: @FreeStuff test If you get a message in the channel you've set up in step one, congratulations, your job here is done! You will from now on get news about new free games in that channel!
Scroll down a bit to see how you can customize the bot's message.
Ever feel lost? Type @FreeStuff help for help!

Customisation

Mention a role

The bot can mention a role whenever there's free stuff! Works nice with so-called self-roles other bots provide. If you want the bot to mention @everyone, check if it has the permissions to do so! @FreeStuff set mention @rolename

Apply themes

You can apply a theme that changes the look of the bot's messages: @FreeStuff set theme 1 The 1 in this case stands for the theme 1. A full list of all available themes can be found here

Change the currency

With some themes the bot also shows the original price. You can choose to show this price either in euros or us-dollars using: @FreeStuff set currency â‚¬ You can also write euro / dollar if you don't have access to these symbols.

Enable auto-reaction

The bot can automatically react with the :free: emoji to each of it's messages. You can toggle this using: @FreeStuff set reaction on/off
